## 前言

本项目主要用于收集prompts、整理短语效果样图、翻译prompts，方便想用AI绘画产出预想结果的人提前了解需要哪些词汇来组织prompts，希望能起到一些词典的效果。

大部分prompts会使用机翻来翻译+少量人工优化，主要是懒。

推荐使用gitbook版阅读。
